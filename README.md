# sxs-tools

## 项目简介
sxs-tools 是一个 Java 工具库，提供了一系列常用的工具类，用于简化日常开发中的常见操作。

## 主要功能
- 金额处理工具
  - 支持金额中文大写转换
  - 支持金额格式化（千分位）
  - 支持金额小写转换
  - 最大支持金额：99999999999999999.99

- 字符串工具
  - 字符串格式化
  - 空值判断
  - 字符串转换

- 安全工具
  - RSA 加解密
  - XSS 防护
  - 数据安全处理
- 。。。

## 使用要求
- JDK 17 或以上
- Maven 3.0 或以上

## 快速开始
1. 添加依赖
```xml
<dependency>
    <groupId>com.suxiaoshuai</groupId>
    <artifactId>sxs-tools</artifactId>
    <version>1.0.0</version>
</dependency>
```
